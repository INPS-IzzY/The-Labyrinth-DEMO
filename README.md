## Room (hand-made but are ordered randomly)
Rooms have 1-3 exits
Randomly generated exits to randomly selected rooms 
The game determines a random tree of rooms such that no room is repeated twice within its branch.


The boss room has only one exit -> next level
The checkpoint room only have one exit -> boss room
The fifth room only has one exit -> checkpoint


Final Room of a Section has a Boss
Final Room of the Final Section has the Final Boss (Minotaur)


Side-on player view like Hollow Knight


Each room has monsters

# - Nudges to talk to Ariadne!!!

If you don't talk to Ariadne, you can still go through the labyrinth, but when you die you go to Hades

If you skip rooms, if you get pulled back to the start via the golden thread, you can still be hit by monsters on the way out

# Hades
Either a long-endurance challenge or a high-difficulty encounter
If you can't get through the challenge, you cannot exit any other way and will have to restart
If you can get through the challenge, you go to back to where you were when you died but with nothing that you weren't wearing
